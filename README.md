# Instrument Tuner

An instrument tuner using your microphone.

## Development

### Setup
```bash
npm install
```

### Build
```bash
npm run build
```

## Output

Production files are in the `dist/` folder:
- `dist/index.html` - Main application
- `dist/assets/` - Bundled assets

## Deploy

Copy the contents of `dist/` to any static hosting service.
